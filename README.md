# shotkeys
Aqui você compra keys aleatórias de jogos para ativar na sua conta Steam ou Ubisoft Connect. As keys são 100% legítimas, adquiridas com fornecedores verificados e entregues de forma rápida!

## Configuração local rápida
1. Copie o arquivo `config.sample.php` para `config.php`.
2. Configure as variáveis de ambiente do banco (recomendado) ou edite o `config.php`.

Variáveis esperadas:
- `DB_HOST`
- `DB_NAME`
- `DB_USER`
- `DB_PASS`

> Dica: não versione arquivos com credenciais reais. O `.gitignore` já ignora `.env`.
